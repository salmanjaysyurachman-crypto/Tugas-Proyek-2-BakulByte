## 🧠 About Bakul Byte

**Bakul Byte** adalah sistem manajemen sederhana untuk membantu UMKM dalam mencatat:

- 📦 Stok Barang  
- 🛒 Transaksi Penjualan  
- 📊 Laporan Harian  

Nama *Bakul Byte* berasal dari:

- **Bakul** → Representasi pedagang kecil / UMKM  
- **Byte** → Simbol transformasi digital  

🎯 **Filosofi:**  
> Menghubungkan pedagang tradisional dengan solusi digital modern.

---

## 🎯 Project Goals

- Membantu UMKM mencatat transaksi secara terstruktur
- Mengurangi kesalahan pencatatan manual
- Memberikan laporan penjualan yang lebih rapi
- Mengembangkan sistem dari CLI menjadi Telegram Bot

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/USERNAME/Bakul-Byte.git
cd Bakul-Byte
```

---

### 2️⃣ Install Dependencies

Pastikan Python 3.10+ sudah terinstall.

Jika menggunakan `requirements.txt`:

```bash
pip install -r requirements.txt
```

Jika belum tersedia:

```bash
pip install python-telegram-bot
```

(SQLite sudah bawaan Python)

---

## ▶️ Running Program

### 🖥️ CLI Version (Project 2)

```bash
python src/main.py
```

Program berjalan melalui terminal.

---

### 🤖 Telegram Bot Version (Project 3)

Pastikan TOKEN sudah diisi pada `config.py`.

```bash
python src/bot.py
```

Bot akan aktif dan siap digunakan melalui Telegram.
