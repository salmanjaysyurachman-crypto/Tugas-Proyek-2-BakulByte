## 🧠 About Bakul Byte

**Bakul Byte** adalah sistem manajemen sederhana untuk membantu UMKM dalam mencatat penjualan, stok barang, dan laporan harian.

Nama *Bakul Byte* berasal dari:
- **Bakul** → Pedagang kecil / UMKM
- **Byte** → Representasi teknologi digital

🎯 Filosofi:
> Menghubungkan pedagang tradisional dengan solusi digital modern.

---

## 🎯 Project Goals

- Membantu UMKM mencatat transaksi secara terstruktur
- Mengurangi kesalahan pencatatan manual
- Memberikan laporan penjualan yang lebih rapi
- Mengembangkan sistem dari CLI menjadi Telegram Bot
